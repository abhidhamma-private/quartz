## 2.소스
1)내 소스
```python
def solution(a, b, n):
    answer = 0
    
    while n >= a:
        mod = n % a # 콜라를 교환하고 남은 빈 병의 개수
        n = (n//a) * b # 교환한 콜라의 개수
        answer += n # 총 콜라 수
        n += mod # 빈병이 된 콜라에 교환하고 남은 빈 병의 개수를 더해서 바꿀때 사용한다
        
    return answer
```
