## 2. 용법
### 1) n진수 -> 10진수
> 
```python
print(int('101',2)) # 5
print(int('202',3)) # 20
print(int('303',4)) # 51
print(int('404',5)) # 104
print(int('505',6)) # 185
print(int('ACF',16)) # 2767
```

### 2)  10진수 -> n진수
>
```python
print(bin(11)) # 0b1011  2진수 1011(2)
print(oct(11)) # 0o13 8진수 13(8)
print(hex(11)) # 0xb 16진수 b(16)

print(bin(11)[:2]) # 1011  진수표시부분 지우려면 [:2]
```

## 3.예제 
> ex) 두 십진수를 비트연산해서 이진수로 변환 
```python
print(bin(9|30)) # 01001(2) | 11110(2) = 11111(2)
```
