## 1. 하는 일
> 숫자를 입력받아 빈 개수 만큼 문자의 앞에 0으로 빈칸을 채움

## 2. 용법
### 1) zfill
> 0으로 채운다
```python
"2".zfill(3) # 002
"50000".zfill(5) # 50000
"123".zfill(5) # 00123
```

### 2) rjust
>원하는 문자로 채운다
```python
"2".zfill(3, "a") # aa2
"50000".zfill(5, "a") # 50000
"123".zfill(5, "0") # 00123
```
